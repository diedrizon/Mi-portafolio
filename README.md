# Portafolio · Diedrizon Fargas

Portafolio personal de un desarrollador Fullstack — rediseñado con un enfoque
moderno y animaciones brutales con **anime.js**.

## ✨ Características

- **Hero animado** con entrada en cascada letra por letra (anime.js) y rol rotativo.
- **Fondo aurora** con blobs de color flotantes, rejilla y grano sutil.
- **Constelación interactiva** en canvas que reacciona al puntero.
- **Cursor personalizado** con anillo magnético.
- **Reveal en scroll** de cada sección con `IntersectionObserver` + anime.js.
- **Tarjetas de proyecto con tilt 3D** y resplandor que sigue al ratón.
- **Contadores animados**, marquee de tecnologías y navbar flotante con sección activa.
- **Botones magnéticos**, barra de progreso de scroll y diseño 100% responsive.
- Respeta `prefers-reduced-motion`.

## 🛠️ Stack

React 19 · Vite · anime.js · react-icons · CSS moderno (glassmorphism, gradientes, `color-mix`).

## 🚀 Scripts

```bash
npm install      # instalar dependencias
npm run dev      # desarrollo con HMR
npm run build    # build de producción
npm run preview  # previsualizar el build
npm run lint     # eslint
```

## 📝 Por personalizar

En `src/App.jsx`, el objeto `SOCIALS` tiene placeholders (`#`) para LinkedIn,
Facebook y TikTok — reemplázalos por tus enlaces reales.
